# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of Image Info Tool
- Batch image information extraction
- GUI interface with egui framework
- File renaming and editing capabilities
- Undo/Redo functionality
- One-click filename clearing
- CMYK/RGB conversion
- Excel (CSV) export
- Thumbnail preview

### Features
- Support for multiple image formats (JPG, PNG, BMP, GIF, TIFF, etc.)
- Parallel processing with Rayon
- Multi-threaded UI with message channels
- Cross-platform support (Windows, macOS, Linux)
- Double-click filename editing
- Right-click context menu
- Keyboard shortcuts (Enter to save, Esc to cancel)
- Smart extension handling
- File conflict resolution
