# 互动点读电子书制作工具 (Interactive E-book Creator)

这是一个强大且易用的互动电子书制作系统，支持从零开始制作包含音频点读功能的互动绘本，并一键发布。

## ✨ 核心功能

### 🛠 编辑器 (Editor)
- **批量导入页面**：一次性选择几十张图片，自动生成电子书页面。
- **批量添加按钮**：一键为页面生成指定数量的点读按钮（例如每页6个），自动网格排列。
- **批量导入音频**：一键导入整个音频文件夹，自动匹配。
- **全局序号系统**：按钮跨页面自动连续编号（1-6, 7-12...），删除按钮后自动重排。
- **所见即所得**：拖拽调整按钮位置，实时预览。
- **一键打包**：生成包含所有资源（图片+音频+配置）的 ZIP 包。

### 📱 播放器 (Player)
- **沉浸式阅读模式**：全屏显示，无干扰，自适应 A4 比例。
- **流畅交互**：
  - **翻页**：支持键盘左右键、鼠标点击左右区域、触摸滑动。
  - **点击播放**：点击按钮播放音频，图标自动切换（▶ ⟷ ⏸）。
- ** ZIP 直读**：直接打开编辑器导出的 ZIP 包，无需解压。

## 📖 使用指南

### 第一步：制作电子书 (Editor)

1. **打开编辑器**：双击 `ui/editor.html`。
2. **导入页面**：
   - 点击左侧边栏底部的 **"批量导入页面"**。
   - 选择你的绘本图片（支持多选）。
3. **添加点读按钮**：
   - 在顶部工具栏的输入框输入数量（例如 `6`）。
   - 点击 **"批量添加"** 按钮。
   - 系统会自动为**所有页面**生成6个均匀分布的按钮。
4. **导入音频**：
   - 点击底部面板的 **"批量导入音频"**。
   - 选择存放音频的文件夹。
   - 系统会自动将 `1.mp3` 匹配给第1个按钮，以此类推。
5. **微调**：
   - 拖动画面上的按钮到正确位置。
   - 可以在左侧点击切换不同页面。
6. **导出发布**：
   - 点击底部的 **"📦 一键打包发布 (ZIP)"**。
   - 保存生成的 `.zip` 文件。

### 第二步：播放与阅读 (Player)

1. **打开播放器**：双击 `ui/index.html`。
2. **加载书籍**：
   - 点击右上角的 **"打开书"** 按钮。
   - 选择刚才导出的 `.zip` 文件。
3. **开始阅读**：
   - **翻页**：按键盘 `→` 或点击屏幕右侧。
   - **点读**：点击画面上的圆圈（▶）播放音频。
   - **退出**：按 `ESC` 键退出全屏/阅读模式。

## 🎨 界面说明

### 按钮状态
- **▶ 实心三角**：等待播放。
- **⏸ 双竖线**：正在播放（点击可暂停）。
- **无图标**：播放结束，自动恢复为三角。

### 阅读操作
| 操作 | 鼠标/触摸 | 键盘 |
| :--- | :--- | :--- |
| **下一页** | 点击右侧 30% / 左滑 | `→` (右方向键) |
| **上一页** | 点击左侧 30% / 右滑 | `←` (左方向键) |
| **暂停/播放** | 点击按钮 | `空格键` |
| **退出** | - | `ESC` |
| **全屏** | - | `F` |

## 🔧 技术细节

- **纯前端架构**：无需服务器，所有逻辑在浏览器中运行。
- **数据格式**：
  - `book.json`: 存储页面结构和按钮坐标。
  - `.zip`: 包含 `book.json` + `images/` + `audio/` 的完整包。
- **自适应布局**：基于通过 CSS `aspect-ratio` 实现的 A4 响应式容器，完美适配 iPad 和 PC。
