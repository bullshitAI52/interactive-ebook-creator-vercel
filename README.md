# 图片信息批量处理工具 (Rust重构版)

这是对原Python程序"显示图片信息终结版1126.py"的Rust重构版本。

## 功能对比

### ✅ 已实现功能
1. **图片信息提取** - 获取像素尺寸、物理尺寸、色彩模式、文件大小
2. **缩略图生成** - 128×128 PNG缩略图
3. **GUI界面** - 使用egui框架，支持图片列表和预览
4. **基本架构** - 模块化设计，易于扩展

### ⚠️ 待完善功能
1. **文件对话框** - 需要集成`rfd`库实现原生文件选择
2. **DPI提取** - 需要集成`exif`库获取准确的DPI信息
3. **CMYK转换** - 需要实现专业的RGB→CMYK转换算法
4. **Excel导出** - 需要完善calamine库的使用
5. **批量重命名** - 需要集成到GUI中
6. **色彩转换** - 需要集成到GUI中

## 项目结构

```
src/
├── main.rs          # 程序入口
├── app.rs           # GUI应用程序
├── image_info.rs    # 图片信息处理
├── utils.rs         # 工具函数
└── excel_export.rs  # Excel导出功能
```

## 运行方法

```bash
# 编译运行
cargo run

# 仅编译
cargo build --release
```

## 依赖说明

- `egui` / `eframe` - GUI框架
- `image` - 图片处理
- `calamine` - Excel文件处理
- `tokio` - 异步支持
- `log` - 日志系统

## 与原Python版本的差异

| 特性 | Python版本 | Rust版本 |
|------|------------|----------|
| GUI框架 | Tkinter | egui |
| 性能 | 中等 | 高（编译优化） |
| 内存安全 | 一般 | 优秀 |
| 跨平台 | 是 | 是 |
| 包大小 | 较小 | 较大（静态链接） |
| 启动速度 | 快 | 快 |
